Theme Options

Here are the options the UM-Plug Plugins provides for your Theme. Remember, that each Wordpress themes may provide a different set of options, or not options at all. If you want to use all of this Theme Options, you must edit your php, styles and scripts according to ours, or use UM Theme Setup as guidances.

Open Sans

Since Wordpress 3.8, Wordpress use Open-Sans Webfont. Check to unload Open-Sans Web font.

Styles and scripts

UM-Plug split the Styles and scripts into:

  1. Normalize
  2. base, Wordpress Default-styles, base on Underscores.me(_s)
  3. UM-GUI, form and styles FX
  4. Colour Schemes, avaliable on customize menu
  5. Layout, avaliable on customize menu

IE html5

Include html5 hack for IE9 and IE8. Uncheck this option if you had your own theories about IE Hack.

UM-GUI

UM-GUI is a styles and scripts that UM Themes required. Uncheck this if you don't want to load/use UM-GUI. UM-GUI contains: ajax wp logins, off-canvassing, dialog box, etc. in small manner
