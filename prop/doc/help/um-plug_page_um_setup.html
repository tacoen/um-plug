UM Theme Setup

This page will help you intergate your theme with/into UM-Plug.

Wordpress Defaults

UM is base on Underscores(_s) Theme. So Our Wordpress Default are using _s template tags, html tags, class and ID.

You could learn about Underscores(_s) Theme at (https://github.com/Automattic/_s)

CSS Reset

The reset using normalize.css v3.0.0 (http://git.io/normalize)

UM-GUI

um-gui-lib provide frameworks for all UM-Themes

Create

You can modified UM scripts and styles, please using Create button, that surely make a copies of the needed files from UM-Plug Plugins directory to your themes directory, and just like all wordpress themes child theme will be able to use parent resources, and template files.
